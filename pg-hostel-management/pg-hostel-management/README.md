# PG Hostel Management System

## 📌 Project Description
This project is a backend API system for PG Hostel Management built using Django. It manages users, rooms, bookings, and authentication.

---

## 🚀 Features
- User Registration & Login (JWT Authentication)
- Role-based access (Student, Owner, Caretaker)
- Room management
- Booking system
- REST API using Django REST Framework

---

## 🛠️ Tech Stack
- Python
- Django
- Django REST Framework
- MySQL / SQLite
- JWT Authentication
